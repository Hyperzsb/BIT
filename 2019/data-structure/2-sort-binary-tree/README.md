# 2. Sort Binary Tree (排序二叉树) 

**Description:**

建立并中序遍历一个排序二叉树

排序二叉树是指左子树的所有节点的值均小于它根节点的值,右子树的所有节点的值均大于它根节点的值,如下图是一棵排序二叉树

**Input:**

输入有一行，表示若干个要排序的数，输入0时停止

**Output:**

二叉树的凹入表示

和二叉树的中序遍历序列

**Example:**

- Input:

```
42 168 35 101 270 125 79 259 263 165 6 246 182 62 192 296 243 28 37 0 
```

- Output:

```
        6
            28
    35
        37

            79

    168
```
