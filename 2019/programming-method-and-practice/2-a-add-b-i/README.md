# 2. A+B I (A+B (I))

**Description:**

学过《计算机科学导论》的你应该熟悉“二进制运算”吧？

和十进制不同的是：二进制运算“逢二进一”。下面举一个二进制加法的运算实例:

```
   11101
+    110
--------

```

下面请你模拟这个过程。
