# 17. Gift Wrap (打包礼物) <font color=RED>(Buggy)</font>

**Description:**

再过一段时间就是龙龙心心念念的她的生日，龙龙准备了 ![k](data:image/svg+xml;charset=utf8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20xmlns%3Awrs%3D%22http%3A%2F%2Fwww.wiris.com%2Fxml%2Fcvs-extension%22%20height%3D%2219%22%20width%3D%2210%22%20wrs%3Abaseline%3D%2215%22%3E%3C%21--MathML%3A%20%3Cmath%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F1998%2FMath%2FMathML%22%20style%3D%22font-family%3A%27Times%20New%20Roman%27%22%3E%3Cmi%3Ek%3C%2Fmi%3E%3C%2Fmath%3E--%3E%3Cdefs%3E%3Cstyle%20type%3D%22text%2Fcss%22%2F%3E%3C%2Fdefs%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2216%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%224.5%22%20y%3D%2215%22%3Ek%3C%2Ftext%3E%3C%2Fsvg%3E) 个礼物送给她，并且决定用顺丰快递寄过去。

但是龙龙发现，因为 ![k](data:image/svg+xml;charset=utf8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20xmlns%3Awrs%3D%22http%3A%2F%2Fwww.wiris.com%2Fxml%2Fcvs-extension%22%20height%3D%2219%22%20width%3D%2210%22%20wrs%3Abaseline%3D%2215%22%3E%3C%21--MathML%3A%20%3Cmath%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F1998%2FMath%2FMathML%22%20style%3D%22font-family%3A%27Times%20New%20Roman%27%22%3E%3Cmi%3Ek%3C%2Fmi%3E%3C%2Fmath%3E--%3E%3Cdefs%3E%3Cstyle%20type%3D%22text%2Fcss%22%2F%3E%3C%2Fdefs%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2216%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%224.5%22%20y%3D%2215%22%3Ek%3C%2Ftext%3E%3C%2Fsvg%3E) 太大了，所以要发很多个快递，这样就要很多的快递费。但是作为一个死肥宅，龙龙还要省点钱去买DELL的显示器、Cherry的机械键盘、NVIDA的最新煤气罐GTX2080呢……所以龙龙感觉钱不够了。

每一个礼物都被包装的明明白白，第 ![i](data:image/svg+xml;charset=utf8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20xmlns%3Awrs%3D%22http%3A%2F%2Fwww.wiris.com%2Fxml%2Fcvs-extension%22%20height%3D%2219%22%20width%3D%226%22%20wrs%3Abaseline%3D%2215%22%3E%3C%21--MathML%3A%20%3Cmath%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F1998%2FMath%2FMathML%22%20style%3D%22font-family%3A%27Times%20New%20Roman%27%22%3E%3Cmi%3Ei%3C%2Fmi%3E%3C%2Fmath%3E--%3E%3Cdefs%3E%3Cstyle%20type%3D%22text%2Fcss%22%2F%3E%3C%2Fdefs%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2216%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%222.5%22%20y%3D%2215%22%3Ei%3C%2Ftext%3E%3C%2Fsvg%3E) 个礼物算上包装的体积就是 ![v 下標 i](data:image/svg+xml;charset=utf8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20xmlns%3Awrs%3D%22http%3A%2F%2Fwww.wiris.com%2Fxml%2Fcvs-extension%22%20height%3D%2226%22%20width%3D%2214%22%20wrs%3Abaseline%3D%2215%22%3E%3C%21--MathML%3A%20%3Cmath%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F1998%2FMath%2FMathML%22%20style%3D%22font-family%3A%27Times%20New%20Roman%27%22%3E%3Cmsub%3E%3Cmi%3Ev%3C%2Fmi%3E%3Cmi%3Ei%3C%2Fmi%3E%3C%2Fmsub%3E%3C%2Fmath%3E--%3E%3Cdefs%3E%3Cstyle%20type%3D%22text%2Fcss%22%2F%3E%3C%2Fdefs%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2216%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%224.5%22%20y%3D%2215%22%3Ev%3C%2Ftext%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2212%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%2210.5%22%20y%3D%2222%22%3Ei%3C%2Ftext%3E%3C%2Fsvg%3E)。龙龙发现，如果两个礼物的体积满足**小礼物的体积的两倍不超过大礼物的体积**，那么小礼物可以勉强塞在大礼物里。同时**一个大礼物里最多只能塞一个其他小礼物，无论小礼物自己内部还有没有更小的礼物**。如果小礼物被塞进大礼物里了，那么龙龙就不用再付小礼物的快递费了。请问龙龙应该如何安排硬塞一下，使得最后剩下的礼物数量最少？请输出方案。

接下来一行共 ![k](data:image/svg+xml;charset=utf8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20xmlns%3Awrs%3D%22http%3A%2F%2Fwww.wiris.com%2Fxml%2Fcvs-extension%22%20height%3D%2219%22%20width%3D%2210%22%20wrs%3Abaseline%3D%2215%22%3E%3C%21--MathML%3A%20%3Cmath%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F1998%2FMath%2FMathML%22%20style%3D%22font-family%3A%27Times%20New%20Roman%27%22%3E%3Cmi%3Ek%3C%2Fmi%3E%3C%2Fmath%3E--%3E%3Cdefs%3E%3Cstyle%20type%3D%22text%2Fcss%22%2F%3E%3C%2Fdefs%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2216%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%224.5%22%20y%3D%2215%22%3Ek%3C%2Ftext%3E%3C%2Fsvg%3E) 个正整数，依次表示这些礼物的体积 ![v 下標 i](data:image/svg+xml;charset=utf8,%3Csvg%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F2000%2Fsvg%22%20xmlns%3Awrs%3D%22http%3A%2F%2Fwww.wiris.com%2Fxml%2Fcvs-extension%22%20height%3D%2226%22%20width%3D%2214%22%20wrs%3Abaseline%3D%2215%22%3E%3C%21--MathML%3A%20%3Cmath%20xmlns%3D%22http%3A%2F%2Fwww.w3.org%2F1998%2FMath%2FMathML%22%20style%3D%22font-family%3A%27Times%20New%20Roman%27%22%3E%3Cmsub%3E%3Cmi%3Ev%3C%2Fmi%3E%3Cmi%3Ei%3C%2Fmi%3E%3C%2Fmsub%3E%3C%2Fmath%3E--%3E%3Cdefs%3E%3Cstyle%20type%3D%22text%2Fcss%22%2F%3E%3C%2Fdefs%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2216%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%224.5%22%20y%3D%2215%22%3Ev%3C%2Ftext%3E%3Ctext%20font-family%3D%22Times%20New%20Roman%22%20font-size%3D%2212%22%20font-style%3D%22italic%22%20text-anchor%3D%22middle%22%20x%3D%2210.5%22%20y%3D%2222%22%3Ei%3C%2Ftext%3E%3C%2Fsvg%3E)。
